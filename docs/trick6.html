Chapter 7 Trick 6 - Create new variables

You might want to create new variables, such as calculating BMI from height and mass. Add as many COMPUTE function below as needed. The command reads as COMPUTE height = ht/100., make a new variable called height by dividing the original variableht by 100. You can either create a new variable or replace the existing variable by using a new name or the original name, respectively.

Math functions

+ Add

- Subtract

/ Divide

* Multiply

** Power

Highlight this command and press the play button.

COMPUTE height = ht/100. 
COMPUTE weight = weight/100.
COMPUTE BMI = weight/(height**2).
EXECUTE.
